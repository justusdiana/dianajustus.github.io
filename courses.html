These are the CIS courses I have taken so far:


CIS 1200 - Intro to Database Systems

This course covered the basics of Database Management Systems. I learned how to create, modify and query a database. I learned about the relational model, SQL through Apex Oracle Workspace, and database design with a datamodeler. No Projects were assigned for this course.



CIS 1300 - Networking Concepts

This course is about how to install, configure and troubleshoot a computer network. It introduces the fundamental building blocks that form a modern network, such as hardware, topologies, and protocols, along with the introduction to the OSI Model. One of the projects I had to do was about giving a recommendation for a network solution for a small company, involving the network topology, and network models.
